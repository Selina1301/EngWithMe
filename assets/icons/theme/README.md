# Theme icon folder

Giai nen bo icon/theme icon vao dung thu muc nay:

```text
assets/icons/theme/
```

Duong dan day du tren may hien tai:

```text
C:\Code\Demo_Study_English\assets\icons\theme
```

Goi y dat ten file de de gan vao HTML/CSS:

```text
assets/icons/theme/favicon.ico
assets/icons/theme/apple-touch-icon.png
assets/icons/theme/icon-192.png
assets/icons/theme/icon-512.png
assets/icons/theme/logo.svg
assets/icons/theme/icons/
```

Neu file ZIP sau khi giai nen tao them mot thu muc long nhau, hay dua cac file icon ra truc tiep trong `assets/icons/theme/`.
